Concepts

The public API is a set of service interfaces which operate on simple value objects. The value objects are abstract classes with public properties and abstract methods for retrieving or setting aggrate objects (where getters may be implemented with lazy loading). For all abstract methods there is an equivalent magic setter or getter. That means that each magic getter or setter is documented by the corresponding method.
